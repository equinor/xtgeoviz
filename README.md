Utility functions for plotting xtgeo objects.

## Install

xtgeoviz is available from PyPI.

```sh
pip install xtgeoviz
```

## Usage

```python
import xtgeoviz.plot as xtgplot
import xtgeo

surf = xtgeo.surface_from_file("somemap.gri")
xtgplot.quickplot(surf)
```
